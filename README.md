# EstadosBrasileiros

## Instalação

Adicionar a gem no Gemfile:

```ruby
gem 'estados_brasileiros', git: "https://github.com/torneseumprogramador/estados_brasileiros_gem.git"
```

Rodar o comado:
```sh
bundle install
```

Rodar o generate:
```sh
rails g estados_brasileiros:install
```

Rodar a rake:
```sh
rake impotacao_estados:importar
```
